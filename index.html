How It Works

1

发现 PDF 文件

在项目目录中搜索 PDF 文件，确认目标论文。

2

提取文本与图片

使用 pdfplumber 提取全文文本，pymupdf 提取所有嵌入图片，自动修复 PDF 文本间距问题。

3

规划文档结构

分析论文结构：识别标题、摘要、章节、图表、公式、参考文献等，规划 HTML 输出格式。

4

生成双语 HTML

Claude Code 全文理解后一次性生成完整的中英对照 HTML，包含 CSS 样式、图片嵌入、表格渲染。
